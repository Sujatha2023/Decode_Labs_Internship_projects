---

## 🎯 Objective

![Target](https://img.icons8.com/color/96/goal.png)

The goal of this project is to clean and prepare raw data using **Python** and **Pandas** by:

✅ Handling missing values  
✅ Removing duplicate records  
✅ Standardizing date formats  
✅ Cleaning text columns  
✅ Improving data quality and integrity

## 🔄 Steps Performed

![Workflow](https://img.icons8.com/color/96/workflow.png)

1. 📥 Loaded Excel dataset using Pandas  
2. 🔍 Identified missing/null values  
3. 🧩 Filled missing CouponCode values  
4. 🧹 Removed duplicate records  
5. 📅 Converted date columns into standard format  
6. ✨ Standardized text formatting  
7. 💾 Saved cleaned dataset into a new Excel file
